# Security Policy

## Reporting a Vulnerability

Security is a top priority for the Void extension. If a vulnerability or potential privacy leak is discovered, please prioritize responsible disclosure.

To report a security issue, please open a new Issue in this repository with the prefix **[SECURITY]** in the title. These reports will be prioritized to ensure user data remains protected.

## Supported Versions

Security updates are currently provided for the latest release series:

| Version | Supported |
| :--- | :--- |
| 0.0.x | Yes |
| < 0.0.1 | No |
